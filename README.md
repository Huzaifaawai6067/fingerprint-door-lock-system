# 🔐 PIC16F877A Fingerprint Authentication System (MATLAB Integration)

## 📌 Overview

This project implements a biometric authentication system using the PIC16F877A microcontroller and a fingerprint sensor, with MATLAB used for data processing, simulation, and algorithm development. The system verifies user identity and controls access through a relay-based locking mechanism.

---

## ⚙️ Features

* Fingerprint-based user authentication
* Microcontroller-based control using PIC16F877A
* MATLAB integration for data processing and algorithm simulation
* Real-time access control using relay module
* Scalable design for embedded security applications

---

## 🛠️ Technologies Used

* PIC16F877A Microcontroller
* MATLAB (signal/data processing & simulation)
* Embedded C (for PIC programming)
* Fingerprint Sensor Module
* Relay Module

---

## 🔌 System Architecture

1. Fingerprint sensor captures user input
2. Data is processed and matched (via MATLAB or embedded logic)
3. PIC16F877A verifies authentication result
4. Relay module is triggered to unlock/lock system

---

## 🚀 How It Works

The system captures a fingerprint and compares it against stored templates. MATLAB is used to simulate or process fingerprint data, while the PIC16F877A executes control logic. Upon successful authentication, the microcontroller activates a relay to grant access.
